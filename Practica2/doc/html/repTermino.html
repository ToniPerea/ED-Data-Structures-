Invariante de la representatción

Se debe cumplir:

  • Un término no puede contener definiciones sino contiene antes una palabra
  • El número de palabras de un término es ninguna o una.
  • El número de definiciones de un término es como mínimo 0, "sin máximo".

Función de abstracción

Palabra -> Objeto string Definiciones -> Array de objetos string uno por cada definición

He elegido este tipo rep, pero he considerado usar también una lista enlazada para las definiciones. Ya que no nos va a hacer falta eliminar ni insertar definiciones, solo añadir, por simplicidad he implementado un vector dinámico para alojar las definiciones.
