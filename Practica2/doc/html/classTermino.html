Descripción detallada

T.D.A. Termino.

El tipo de datos abstracto término, representa una relación entre una palabra sus significados o definiciones. Puede estar vacio (no tener palabra que representar)

Este estado se puede consultar por el método vacio().

Además, puede contener una palabra pero no definiciones.

Un término se representa como una entrada de diccionario: Palabra: 0 ... 1 Definiciónes 0 ... n

Se almacena almacena su palabra y definiciones en memoria dinámica, es mutable.
